青龙使用教程

一、安装

选择「依赖管理」

新建 Linux 依赖

1. 点击 Linux

2. 点击新建依赖

3. 输入依赖

gcc g++ make libffi-dev openssl-dev

4. 点击确认，等待安装完成

新建 Python 依赖

1. 点击 Python

2. 点击新建依赖

3. 输入 dailycheckin

4. 点击确认，等待安装完成

二、配置

⚠️

请务必到 http://www.json.cn (opens in a new tab) 网站检查 config.json 文件格式是否正确！

选择「脚本管理」

点击 ➕ 添加

输入文件名 config.json 后确定

配置 config.json 文件

参考配置说明文档 ，并修改 config.json

三、运行

  1. 运行全部脚本
  2. 运行指定脚本（包含），可以同时选择多个，用「空格」分开
  3. 运行指定脚本（排除），可以同时选择多个，用「空格」分开
