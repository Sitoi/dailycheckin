小米运动

配置示例

config.json
{
  "MIMOTION": [
    {
      "max_step": "20000",
      "min_step": "10000",
      "password": "Sitoi",
      "phone": "18888xxxxxx"
    },
    {
      "max_step": "多账号 最大步数填写，请参考上面",
      "min_step": "多账号 最小步数填写，请参考上面",
      "password": "多账号 密码填写，请参考上面",
      "phone": "多账号 手机号填写，请参考上面"
    }
  ]
}

配置说明

参数说明
MIMOTION.phone小米运动刷步数的「手机」或「邮箱」
MIMOTION.password小米运动刷步数的账号密码
MIMOTION.min_step小米运动刷步数的最小步数
MIMOTION.max_step小米运动刷步数的最大步数
