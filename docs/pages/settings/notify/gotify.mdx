# GOTIFY

### 配置示例

```json filename="config.json" copy
{
  "GOTIFY_URL": "",
  "GOTIFY_TOKEN": "",
  "GOTIFY_PRIORITY": ""
}
```

|         参数          |                     说明                      |
| :-------------------: | :-------------------------------------------: |
|   _**GOTIFY_URL**_    |          [GOTIFY] ,填写 `GOTIFY_URL`          |
|  _**GOTIFY_TOKEN**_   |              填写 `GOTIFY_TOKEN`              |
| _**GOTIFY_PRIORITY**_ |            填写 `GOTIFY_PRIORITY`             |
|   _**MERGE_PUSH**_    | **true**: 将推送消息合并；**false**: 分开推送 |
