# Guten Email Notification (邮箱)

官网: https://github.com/gutenye/email-notification/blob/main/src/templates/DailyCheckIn.md

### 配置示例

```json filename="config.json" copy
{
  "NTFY_URL": "",
  "NTFY_TOPIC": "",
}
```

|        参数         |                                      说明                                       |
| :-----------------: | :-----------------------------------------------------------------------------: |
|   _**NTFY_URL**_    |  例: `https://YOUR_DOMAIN.workers.dev`  |
|  _**NTFY_TOPIC**_   |  例: `YOUR_API_KEY?title=DailyCheckIn`                       |
|  _**MERGE_PUSH**_   |  **true**: 将推送消息合并；**false**: 分开推送                  |
